Gameplay

Galactic Emperor: Hegemony is a strategic game of space warfare which contains some roleplaying elements.

The main, strategic, game is fought within a simple java interface which allows ships to hop from planet to planet, securing resources and skirmishing with the enemy. Starting orders and standing orders allow Overlords to leave commands for their troops which will be executed as required. The gameplay is very simple, operated entirely from a web interface, but allows for complex strategy.

The strategic game is supplemented by Abode, a Skotos Chat Theatre. It is a place for arbitration and negotiation, detailed in prose. It is built on the Skotos StoryBuilder Server and contains ample support for social activities.
