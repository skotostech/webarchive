Artists

The artwork for Grendel's Revenge is by an assortment of artists. Stephanie Law created the evocative cover painting. Nate Mericle created the gorgeous character/caste portraits used throughout the game. FreeFrog Designs created the in-game icons, some of the client UI art, as well as the web portal. Skotos' Michael Blum is responsible for the many colorful terrain maps in Grendel's Revenge. The folks at Worlds Apart created the client UI and functionality.
