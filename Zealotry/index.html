Zealotry - The Next Generation Skotos Client

To give Zealotry a try, click on the XPI below. You will need to give permission in order to install the XPI. As with all Firefox extensions you'll need to restart Firefox for it to be activated.

To connect to a character you can either type zealotry:<character>@<game> in the URL bar or you can go to the appropriate portal page and it should be auto-updated to point to Zealotry upon detection.

If you find any bugs, file a task on any of the games.

Example URL:

  • zealotry:billy@cm

The relevant links:

  • Stable version: zealotry-current.xpi
  • SVN source (slightly outdated): code.is-here.com/zealotry

Good luck!
